# Chefio Backend

Chefio is a social media app focused on food and recipes. This is the backend service built with **Node.js**, providing RESTful APIs for user authentication, recipe management, likes, follows, and real-time notifications via Firebase Cloud Messaging (FCM).

---

## Features

- User signup with email verification code
- JWT-based authentication and authorization
- Add, update, delete, and view recipes
- Like/unlike recipes
- Follow/unfollow chefs (users)
- Push notifications (FCM) for likes, follows, new recipes
- Public Postman API documentation
- Clean MVC architecture & modular codebase

---

## Tech Stack

- **Node.js**, **Express.js**
- **MongoDB** & **Mongoose**
- **JWT** for auth
- **Firebase Cloud Messaging (FCM)**
- **Postman** for API testing and docs
- **Vercel** for deployment
